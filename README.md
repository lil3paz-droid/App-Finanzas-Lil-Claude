# Dashboard Financiero Empresarial

Dashboard interactivo para el control y visualización de datos financieros empresariales.

## 🚀 Características

- 📊 Paneles interactivos desplegables
- 💰 Seguimiento de ingresos y gastos
- 📈 Indicadores de rendimiento (KPI)
- 🎨 Diseño moderno con efectos glassmorphism
- 📱 Responsive design
- ⚡ Carga rápida y optimizada

## 🎯 Demo

[🔗 Ver Dashboard en Vivo](https://tu-usuario.github.io/dashboard-financiero/)

## 📋 Funcionalidades

### Paneles Disponibles
- **💰 Ingreso Total**: Cobrado y por cobrar
- **💸 Gastos**: Fijos y variables
- **📊 Rendimiento**: KPIs y métricas
- **👥 Recursos Humanos**: Nómina y personal
- **📈 Ventas**: Análisis de ingresos por ventas

## 🛠️ Tecnologías

- **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
- **Iconos**: Font Awesome 6.0
- **Efectos**: Glassmorphism, Animaciones CSS
- **Diseño**: Grid Layout, Flexbox

## 📱 Responsive Design

Optimizado para:
- 🖥️ Desktop (1200px+)
- 💻 Laptop (768px - 1199px)
- 📱 Tablet (481px - 767px)
- 📱 Mobile (hasta 480px)
